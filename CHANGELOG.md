# 更新日志

本文档记录项目对外发布的重要变更。

格式参考 Keep a Changelog，并结合当前项目实际情况维护。

## [0.1.0] - 2026-03-31

### 新增

- 提供基于 Next.js 的 Web 前端
- 提供基于 NestJS 的后端服务
- 提供基于 Electron 的桌面端前端壳
- 支持房间创建、加入、最近房间恢复
- 支持播放队列、歌单、聊天与实时同步
- 支持桌面端文件选择、外链打开与版本读取

### 调整

- 桌面端生产模式改为默认加载 `https://witw.top`
- 桌面安装包不再内置本地 Next.js runtime
- 桌面安装包不再内置本地后端服务

### 修复

- 修复 Windows 下 Electron 打包流程
- 修复 `electron-builder` 与 pnpm 工作区下的构建问题
- 修复桌面端发布产物清理与重复打包问题

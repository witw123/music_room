# 歌单能力现状

最后更新：`2026-07-23`

## 当前结论

歌单后端能力和 Web 工作区入口均已接入，当前房间体验仍以共享队列为主。

保留能力：

- 从当前房间队列保存为歌单
- 重命名歌单
- 删除歌单
- 将歌单重新导回房间队列
- 在房间外整理本地歌单和网络歌单
- 浏览网易云/QQ 音乐歌单和专辑，并将曲目导入个人曲库

## 当前 UI 状态

- 房间工作区默认显示 `Library`、`My Playlists`、`Members`
- `/app/playlists` 是房间外的歌单整理入口
- 歌单不参与媒体源选择或成员间资产传输

## 产品优先级

当前版本优先保证：

- 多人同步播放稳定性
- 单一 Segmented Opus/WebRTC 媒体链路
- 个人上传曲库在浏览器刷新后的恢复
- 成员与媒体诊断
- 响应式 Web 工作区和移动浏览器音频体验稳定

## 当前边界

- 歌单作为房间外的整理能力
- 房间内仍以共享队列为主
- 保留“从队列保存歌单”和“把歌单导入队列”的双向转换
- 不把歌单设计成缓存下载、媒体源切换或成员间音频共享入口

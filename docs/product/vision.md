# 产品愿景

## 目标用户

- 想和朋友远程一起听歌的小群体
- 需要临时音乐房进行点歌和同播的社交场景
- 希望保留本地音频主权、不想把文件上传到平台的用户

## 产品定位

Music Room 不是公共版权曲库，也不是服务端流媒体平台。它的价值在于：

- 用极低的服务端媒体成本完成多人同播
- 把房间、播放控制、歌单协作和 P2P 传输组合成一个闭环
- 让本地音乐也能有多人社交体验

## 首版范围

- Web 端房间与播放器
- 游客会话 + 房间码加入
- 本地歌曲导入与元数据解析
- 房间共享队列与点歌
- P2P 分片传输与本地缓存
- 强管理歌单与协作

## 首版不做

- 正版曲库接入
- 服务端音频存储或转发
- 移动端完整体验承诺
- 房主迁移
- 推荐算法和公共发现页
